JavaScript 6

ภาษา JavaScript ในเวอร์ชั่นที่ 6 นี้ได้เพิ่มความสามารถเข้ามามากมายทำให้การเขียนง่ายและมีประสิทธิภาพมากยิ่งขึ้นโดยเฉพาะการเขียนแบบ Functional Programming


JS6 ภาษาเดียว!

  • สร้าง Website ได้
  • นำไปเขียน Mobile app ได้
  • เขียน Game ก็ยังได้
  • หรือ จะนำไปเขียน Desktop app ก็ทำได้

Node.js

Node.js ไม่ใช่ภาษา!! แต่เป็น cross-platform runtime environment หมายความว่า node.js สามารถรัน javascript aplication บน OS ต่างๆได้


Node.js ทำอะไรได้บ้าง

  • สร้าง Server ขึ้นมาได้เลย
  • นำไปเขียน Realtime app เช่น Chatapp, Dashboard, Online game
  • เขียน API ก็ยังได้
  • ที่สำคัญสุดๆ เขียน Website ได้ไวมากๆ

AngularJS

สุดยอด Front-End framework จาก Google ที่จะช่วยทำให้คุณพัฒนาเว็บและแอพได้ล้ำสุดๆ!!



AngularJS ทำอะไรได้บ้าง

  • สร้าง Single Page App ได้ง่ายมากๆ
  • ลืมการ Refrash ไปได้เลยเพราะ AngularJS async รัวๆเลย
  • ทำงานร่วมกับ API นี่จุดเด่นเลย
  • ที่สำคัญเขียน AngularJS มันสนุกมากๆเลย

Bootstrap 3

HTML, CSS, และ JS Framework ที่ฮิตที่สุดในขณะนี้สำหรับการทำ Responsive website ช่วยให้เหล่านักพัฒนาทำงานกันได้ง่ายขึ้นมากๆ



ทำไมต้อง Bootstrap 3

  • เพราะเราต้องการทำ Responsive website แบบสบายๆ
  • เพราะเราต้องการเขียนให้รองรับ Mobile devices
  • เพราะเราต้องการ Productivity ระดับสูง!
  • และเพราะว่า Bootstrap Save Time ได้มากมาย

Ionic Framework

สร้าง Mobile application แบบรวดเร็วทันใจด้วย web technology ที่คุณชื่นชอบ ทั้ง iOS และ Android



Hybrid App ดียังไง

  • Code เดียว ใช้ได้ทั้ง iOS และ Android
  • ประหยัด งบ และ เวลา ที่ใช้ในการพัฒนา
  • Lerning curve ต่ำมาก เพราะไม่ต้องเรียนรู้ภาษาใหม่
  • และแน่นอนว่า Ionic ฟรี ไม่มีค่าใช้จ่าย

MongoDB

NoSQL ยอดฮิตที่ใช้งานง่าย ประสิทธิภาพสูง ขยายฐานข้อมูลได้ ที่สำคัญเร็วมาก



ทำไมต้อง NoSQL

  • แน่นอนว่าเรื่อง ค่าใช้จ่ายในการ scale ประหยัดไปมาก
  • NoSQL จะทำให้คุณสนุกมากกว่าเก่า
  • ถ้าคุณชอบ oop NoSQL เหมาะกับคุณมาก
  • ประสิทธิภาพ ที่จะไม่ทำให้คุณผิดหวัง
